# Online-Bike-Rental-Management-System

A Java-based web application for managing bike rentals through dedicated user and administrator portals, developed as a university group project.

##  Project Overview

The Online Bike Rental Management System was designed to simplify the bike rental process by providing a centralized platform for customers and administrators.

Customers can register, browse bikes, rent bikes, track rentals, manage inquiries, and update their profiles. Administrators can efficiently manage bike inventory, review rental requests, and oversee system operations through an administrative dashboard.

The project follows the **Model-View-Controller (MVC)** architecture and applies **Object-Oriented Programming (OOP)** principles to improve maintainability and code organization.

#  Features
## Customer Portal

- User registration and login
- Secure authentication
- Browse available bikes
- Filter bikes by category
- Submit hourly or daily rental requests
- View active rentals
- View rental history
- Cancel active rentals
- Submit customer inquiries
- Manage user profile
- Change password
- Delete account

## Administrator Portal

- Administrator login
- Dashboard overview
- Manage bike inventory
- Add new bikes
- Update bike details
- Delete bikes
- Approve or reject rental requests
- View customer information
- Manage rental requests

#  Key Highlights
